# Setup

## Skills
Please run the following command to install required skills.
```shell
# Obsidian
npx skills add https://github.com/kepano/obsidian-skills --all
# Shadcn UI
npx skills add shadcn/ui --all
# CopilotKit
npx skills add CopilotKit/CopilotKit/skills --all
```

## Obsidian
Please copy ".obsidian" folder from project root of https://github.com/Kita12123/dev-ctx to this project root directory.
